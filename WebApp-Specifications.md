# Small Web Application Specifications

## Application Description
A small business web application that allows customers to browse products and make purchases online.

## Compute Requirements
- 1 Virtual Machine
- 2 vCPU
- 4 GB RAM
- Linux Ubuntu OS

## Storage Requirements
- 100 GB object storage

## Network Requirements
- 500 GB monthly outbound data transfer

## User Load
- 500–1000 active users per month
