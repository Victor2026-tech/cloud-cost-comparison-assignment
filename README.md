# Cloud Cost Comparison: AWS vs Azure

## Executive Summary

This project compares the cost of hosting a small web application using AWS and Azure across compute, storage, database, and networking services.

## Service Equivalency Mapping

| Function | AWS Service | Azure Service |
|----------|------------|---------------|
| Compute | EC2 | Virtual Machine |
| Object Storage | S3 | Blob Storage |
| Database | RDS MySQL | Azure SQL Database |

## Key Findings

- AWS was more cost-effective for Linux workloads.
- Azure provides advantages for organizations using Microsoft technologies.
- Database pricing and regional selection can significantly affect total cost.
